# lpsolve

A PHP extension to execute [lpsolve](https://sourceforge.net/projects/lpsolve/), a Mixed Integer Linear Programming (MILP) solver.

Compatible with PHP >= 7.0


## How to compile

```bash
apt-get install liblpsolve55-dev php7.4-dev

cd src
phpize
./configure
make
```
